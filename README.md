# Codex Oribis v1.0

This repository publishes **Codex Oribis v1.0** as a **frozen canonical framework** and a minimal reference implementation (**ORIBIS-CORE / MV-CORE**) for deterministic structural validation.

## Status

- Version: **1.0**
- Canonical State: **Frozen**
- Modification: **Restricted**

No changes to primitives, axioms, or invariants are permitted within the v1.x series.

## Scope

Oribis is a **non-verbal relational framework** that evaluates whether a system, rule, or governance structure can exist without internal contradiction under fixed constraints.

Oribis validates **structure only**.

## Non-Scope

Oribis does **not**:
- interpret intent or meaning
- recommend actions or policies
- replace legal, regulatory, or governance processes
- certify correctness, safety, or compliance
